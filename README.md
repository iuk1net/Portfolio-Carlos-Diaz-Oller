## Objetivo
Presentar de forma clara y profesional mi perfil como desarrollador, demostrando capacidad para diseñar, desarrollar y desplegar una aplicación web completa siguiendo MVC, buenas prácticas y seguridad.

## Nota sobre el repositorio público
Este repositorio es público para poder enlazarlo desde LinkedIn. Por seguridad:
- No se deben publicar credenciales ni secretos (usuarios, contraseñas, tokens, cadenas de conexión reales).
- La configuración sensible debe ir en variables de entorno o en ficheros ignorados por Git.
